Built by the students of Lokmanya Tilak College of Engineering

The "Emotion Visualizer" is a web application designed by the students of Lokmanya Tilak College of Engineering to detect and visualize human emotions in real-time using facial recognition techniques.

Programmed using Python

Python has many libraries and frameworks designed specifically for machine learning, such as Flask and PyTorch. These libraries can help speed up development and reduce the need to write repetitive code.

Real-time extraction using OpenCV

developed by Intel, OpenCV is a free cross-platform computer vision library for real-time image processing. OpenCV makes it easy for businesses to utilize and modify the code.

Flask for Backend

Flask is a web application framework written in Python. Flask is based on the Werkzeug WSGI toolkit and Jinja2 template engine.flask provides you with tools, libraries that allow you to build a web application.

hugging-face for model reference

Hugging Face provides the tools to create and train ML models for natural language processing (NLP) and other tasks. Users can work collaboratively with other people in the ML ecosystem
